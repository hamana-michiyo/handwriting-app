1. **モジュール間データ構造統一の重要性**: 異なるプロセッサ間でのキー名統一が不可欠

3. **段階的デバッグの効果**: 詳細ログによる問題箇所の迅速特定
4. **画像データ処理の複雑さ**: bytes（ストレージ用）とarray（shape確認用）の使い分け
